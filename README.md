# Arduino RTTY Keyer

An RTTY Keyer for Arduino Uno.

Use Arduino IDE to build and upload it to your Arduino Uno devices.

Use serial command to control it. for help, type `HELP`.

Please change the `FSK_PIN` , `FSK_MARK` , `FSK_SPACE` , `PTT_PIN` , `PTT_ON`, `PTT_OFF` to a value suitable for use on your radio equipment. the default value is for the Icom IC-7300.

Please change `CALLSIGN` , `CQZ` , `AGE`.
